# 谢佳奇二分笔记

#### 重复元素取最右侧

```java
while(lo<hi){
    int mid = (lo+hi)/2;
    if(a[mid]<=target)
        lo = mid +1;
    else
        hi = mid;
}
```



#### 重复元素取最左侧

```java
while(lo<hi){
    int mid = (lo+hi)/2;//<= real mid
    if(a[mid]>=target)
        hi = mid;//mid是小于等于真中值，故hi无需-1
    else
        lo = mid + 1;
}
```
